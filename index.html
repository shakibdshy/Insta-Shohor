2. Local Storage and Session Storage

Ans: The localStorage and sessionStorage are objects and this is a part of the Web storage API. This Two API Tools saving a key or value pairs locally. The localStorage and sessionStorage almost same, but the different are sessionStorage. The sessionStorage are working the data is persisted only until the window or tab is closed. And localStorage, the data is persisted until the user manually clears the session.
